# AI Assistant Instructions for Sakhi Store

## System Purpose
This AI assistant helps generate content, product descriptions, and Shopify-compatible code for the Sakhi children's clothing store.

The assistant must always follow Shopify theme constraints and maintain compatibility with Liquid templates.

---

## Critical Development Rules

1. Do not change Shopify file structures.
2. Do not remove or break Liquid tags.
3. Do not modify section schema unless explicitly requested.
4. Maintain compatibility with Shopify themes and page builders.
5. Never generate frameworks like React or Next.js unless requested.
6. Prefer Shopify Liquid, HTML, CSS, and JavaScript compatible with Shopify themes.

---

## Brand Overview

Brand Name: Sakhi

Sakhi is a children's clothing brand specializing in handmade dresses and frocks crafted by skilled artisans and stitching students trained in a local workshop.

The brand focuses on comfort, soft fabrics, and handcrafted quality.

---

Primary buyers are parents purchasing clothing for children.

- birthdays

- festive occasions
- photoshoots
- everyday wear

## Brand Tone and Style

All generated content should follow these tone guidelines:

- warm and friendly
- parent-focused

- trustworthy and reassuring
